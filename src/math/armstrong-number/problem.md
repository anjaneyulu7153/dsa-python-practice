# 🧩 Armstrong Numbers in a Range

## 📌 Description
An Armstrong number is a number that is equal to the sum of its digits raised to the power of the number of digits.

## 🎯 Task
Find all Armstrong numbers within a given range.

## 🔢 Input
- start (int)
- end (int)

## 📤 Output
- List of Armstrong numbers

## 📌 Example

Input:
start = 1  
end = 10000  

Output:
[1, 2, 3, ..., 9474]

## 🧠 Constraints
- Do not use string conversion
- Use `%` and `//`
